Подробное описание

Класс компилятора

Дата
Июль 2020 года
Ошибка:
Иногда некорректно преобразует исходный код в абстрактное синтаксическое дерево. Пока не понятно, в каких случаях возникает ошибка и как ее исправить.

Основной класс компилятора, созданный для предоставления пользователю удобного api для компиляции/сборки программ. Также хранит в себе исходный и скомпилированный код программ и параметры компиляции.

Функция добавления нового файла в список исходников

Аргументы

Расширяет массив имен исходников на 1 элемент и копирует в этот элемент имя исходника. Так же увеличивает счетчик исходников на 1.

◆ mode()

short int Compiler::mode ( )

Проверка режима работы компилятора

Возвращает код ошибочного запуска если не передан ни один параметр, не передан ни один исходник или передан неверный параметр. Информация о корректности ключей сохраняется в поле correct_usage в процессе конструирования объекта.
