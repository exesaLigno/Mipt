\doxysubsection{Подробное описание}
Класс компилятора 

\begin{DoxyDate}{Дата}
Июль 2020 года 
\end{DoxyDate}
\begin{DoxyRefDesc}{Ошибка}
\item[\mbox{\hyperlink{bug__bug000001}{Ошибка}}]Иногда некорректно преобразует исходный код в абстрактное синтаксическое дерево. Пока не понятно, в каких случаях возникает ошибка и как ее исправить.\end{DoxyRefDesc}


Основной класс компилятора, созданный для предоставления пользователю удобного api для компиляции/сборки программ. Также хранит в себе исходный и скомпилированный код программ и параметры компиляции.
